interface AdaptationResult {
  studentType: AdaptationType
  content?: AdaptationContent

      type: string

  }
  progress: number
  status: 'pending' | 'processing' | 'completed' | 'error' | 'published'

    has_audio: boolean
    has_video: boolean
    has_images: boolean
    has_practice: boolean

      priority?: 'low' | 'medium' | 'high'

}

## 🚀 Как использовать

### 1. Загрузка уроков из БД

```tsx
const loadLessonsFromDB = async (courseId: string): Promise<Lesson[]> => {
  const { data: lessonsData, error: lessonsError } = await supabase
    .from("course_lessons")
    .select("*")
    .eq("course_id", courseId)
    .order("order_index", { ascending: true })

  return lessonsData.map((lesson: any) => ({
    id: lesson.id,
    title: lesson.title,
    description: lesson.description || "",
    blocks: lesson.blocks || [],
    materials: [],
    tests: []
  }))
}
```

    if (response.ok) {

## ⚠️ Важные замечания

1. **Загрузка уроков**:
   - Уроки загружаются из таблицы `course_lessons`
   - Уроки сортируются по `order_index`
   - Первый урок выбирается по умолчанию

2. **Загрузка адаптаций**:
   - Адаптации загружаются из таблицы `lesson_adaptations`
   - Адаптации загружаются для всех типов (visual, auditory, kinesthetic)
   - Метаданные адаптации загружаются из таблицы `lesson_adaptation_metadata`

3. **Режим "Оригинал"**:
   - Режим "Оригинал" по умолчанию при загрузке страницы
   - Режим "Оригинал" показывает оригинальный контент автора
   - Оригинальный контент загружается из таблицы `course_lessons`

4. **Сохранение изменений**:
   - Изменения сохраняются через API `/api/lesson-adaptation` (метод PUT)
   - Статус адаптации обновляется на 'edited'
   - Версия адаптации увеличивается

5. **Публикация адаптации**:
   - Адаптация публикуется через API `/api/lesson-adaptation/publish` (метод POST)
   - Статус адаптации обновляется на 'published'
   - Публикованные адаптации доступны для студентов

6. **Рекомендации по материалам**:
   - Рекомендации загружаются из таблицы `lesson_adaptation_metadata`
   - Рекомендации показываются только для автора
   - Рекомендации не блокируют публикацию курса
