## 📖 С чего начать?

### Вариант 1: Пошаговая инструкция (рекомендуется)

Откройте файл **ИНСТРУКЦИЯ_РАЗВЕРТЫВАНИЯ.md** и следуйте шагам.

### Вариант 2: Автоматический скрипт

1. Загрузите проект в GitHub
2. На сервере выполните:
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/ruta-education.git /var/www/ruta-education
   cd /var/www/ruta-education
   chmod +x scripts/deploy.sh
   sudo ./scripts/deploy.sh
   ```
3. Настройте `.env.production`
4. Выполните миграцию БД

## 🆘 Если что-то пошло не так

1. Проверьте логи: `pm2 logs ruta-education`
2. Проверьте переменные: `cat /var/www/ruta-education/.env.production`
3. Проверьте статус: `pm2 status`
4. Проверьте Nginx: `systemctl status nginx`
