## 🔍 Структура промптов

### Базовый промпт

```typescript
const basePrompt = `
Ты - эксперт по адаптации образовательного контента для [тип].

ИСХОДНЫЙ КОНТЕНТ УРОКА:
Название: [название]
Описание: [описание]
Блоки урока:
[блоки урока]

ЗАДАЧА: Адаптировать этот контент под тип восприятия "[тип]", создав 5 блоков с трехслойной структурой.

═══════════════════════════════════════════════════════════════════════════════
КРИТИЧЕСКИ ВАЖНО: ПРИНЦИП ДУБЛИРОВАНИЯ С УСИЛЕНИЕМ
═══════════════════════════════════════════════════════════════════════════════

Система НЕ удаляет текст автора. Адаптация = структурирование + добавление специализированных элементов.

ПРИНЦИП РАБОТЫ:
1. Подводка (intro) - короткое вступление (2-3 предложения)
2. Улучшенная текстовая версия (content) - ПОЛНЫЙ контент автора, структурированный
3. Адаптированный элемент (adaptation) - специализированный формат, который ДУБЛИРУЕТ и УСИЛИВАЕТ текст

ПРАВИЛА:
- Текстовая версия должна содержать ВЕСЬ контент автора
- Адаптированный элемент должен отражать те же концепции, что и текст
- Адаптация должна быть конкретной и детальной
- Все ключевые термины должны присутствовать в тексте
`
```

## 🚀 Как использовать

### 1. Генерация адаптации

```typescript
const response = await fetch('/api/ai-adaptation', {
  method: 'POST',
  body: JSON.stringify({
    lessonContent: {
      title: 'Название урока',
      description: 'Описание урока',
      blocks: [
        {
          title: 'Блок 1',
          content: 'Контент блока 1',
          type: 'text'
        }
      ]
    },
    studentType: 'visual',
    lessonId: 'lesson-id',
    saveToDatabase: true
  })
})
```

### 2. Структура ответа

```typescript
{
  success: true,
  adaptedContent: {
    block1: {
      intro: {
        text: "Подводка из 2-3 предложений",
        type: "intro"
      },
      content: {
        title: "Заголовок блока",
        text: "ПОЛНЫЙ текст контента автора, структурированный",
        sections: [
          {
            title: "Подзаголовок",
            content: "Содержание",
            highlighted: ["ключевые", "термины"]
          }
        ],
        type: "text"
      },
      adaptation: {
        type: "visual",
        element: {
          type: "diagram",
          data: {
            nodes: [...],
            connections: [...]
          },
          description: "Описание схемы"
        }
      }
    },
    // ... блоки 2-5
  }
}
```
