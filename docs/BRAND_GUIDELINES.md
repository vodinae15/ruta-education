# 🎨 **Брендбук Ruta Education**

## 📋 **Обзор бренда**

**Ruta Education** — образовательная платформа с двойной персонализацией, которая адаптирует коммуникацию под стиль преподавателя и тип восприятия каждого ученика.

### **Миссия:**
Создать максимально эффективную образовательную среду через персонализацию контента и адаптацию под индивидуальные особенности пользователей.

### **Ценностные предложения:**
- **Персонализация:** 72 варианта подачи материала
- **Простота:** Создание курса за 15 минут
- **Эффективность:** Адаптация под 8 типов авторов и 9 типов учеников
- **Профессионализм:** Научный подход к образованию

## 📝 **Типографика**

### **Шрифтовая система:**
- **Основной шрифт:** Inter (Google Fonts)
- **Моноширинный:** ui-monospace, SFMono-Regular, Monaco, Consolas

### **Иерархия размеров:**

#### **Заголовки:**
- **H1 (Hero):** `text-4xl lg:text-6xl` (36px → 60px)
  - **Вес:** `font-bold`
  - **Использование:** Главный заголовок страницы

- **H2 (Секции):** `text-3xl lg:text-4xl` (30px → 36px)
  - **Вес:** `font-bold`
  - **Использование:** Заголовки секций

- **H3 (Подсекции):** `text-2xl lg:text-3xl` (24px → 30px)
  - **Вес:** `font-bold`
  - **Использование:** Заголовки подсекций

#### **Текст:**
- **Large:** `text-xl lg:text-2xl` (20px → 24px)
  - **Вес:** `font-medium`
  - **Использование:** Важные описания

- **Base:** `text-lg` (18px)
  - **Вес:** `font-normal`
  - **Использование:** Основной текст

- **Small:** `text-sm` (14px)
  - **Вес:** `font-normal`
  - **Использование:** Вторичный текст, карточки

### **Правила типографики:**
- **Межстрочный интервал:** 1.5-1.6
- **Длина строки:** 45-75 символов
- **Отступы между заголовками:** 1.5-2x размер шрифта
- **Контраст:** минимум 4.5:1 для основного текста
