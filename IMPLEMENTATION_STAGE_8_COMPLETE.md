## ⚠️ Важные замечания

1. **Практические задания**:
   - Практические задания МОГУТ адаптироваться под типы восприятия
   - Содержание практических заданий может изменяться в зависимости от типа восприятия
   - Формат выполнения задания адаптируется под тип восприятия

2. **Итоговые тесты**:
   - Итоговые тесты НЕ адаптируются под типы восприятия
   - Содержание теста (вопросы, ответы, правильные ответы) НЕ изменяется
   - Только подача теста может адаптироваться (озвучка текста для аудиалов)

3. **Озвучка тестов**:
   - Озвучка применяется только к тексту теста
   - Вопросы и ответы остаются теми же
   - Озвучка доступна только для аудиального типа восприятия

4. **Отображение тестов**:
   - Тесты отображаются в режиме "Оригинал" из элементов блоков урока
   - Тесты отображаются в адаптированном контенте (блок 5) из оригинального контента
   - Тесты отображаются одинаково для всех режимов

5. **Промпты ИИ**:
   - Обновлены промпты ИИ для разделения практики и тестов
   - Добавлены инструкции о том, что практические задания МОГУТ адаптироваться
   - Добавлены инструкции о том, что итоговые тесты НЕ адаптируются

6. **Компоненты**:
